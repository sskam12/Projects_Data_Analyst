# Анализ рынка недвижимости CIAN (Python)

**О проекте**

В рамках проекта был проведен исследовательский анализ данных объявлений о продаже квартир с платформы CIAN. 

**Проблема**

Стоимость квартиры формируется под влиянием множества факторов: района, площади, этажа, количества комнат, характеристик дома и типа продавца. Для покупателей, продавцов и агентств недвижимости важно понимать, какие из этих факторов действительно оказывают значимое влияние на цену за квадратный метр.
Цель проекта — выявить ключевые факторы ценообразования на рынке недвижимости и проверить статистические гипотезы о влиянии характеристик квартиры на ее стоимость.

**Решение**

В этом проекте проведен полный цикл анализа данных объявлений CIAN:
- выполнена очистка и подготовка данных;
- исследованы распределения и взаимосвязи между признаками;
- проверены статистические гипотезы о влиянии характеристик квартиры на стоимость квадратного метра;
- построена линейная регрессионная модель, позволяющая количественно оценить влияние различных факторов на цену.

В результате удалось определить наиболее значимые факторы ценообразования и подтвердить, какие гипотезы находят статистическое подтверждение.

**Используемый стек**
- Python
- pandas
- numpy
- matplotlib
- seaborn

- statsmodels

**Этапы работы**

1. Подготовка данных:

- удаление дубликатов;
- обработка пропущенных значений;
- удаление нерелевантных признаков;
- заполнение части пропусков на основе имеющейся информации;
- преобразование категориальных признаков;
- создание dummy-переменных.

2. Исследовательский анализ данных (EDA)

В ходе анализа были рассчитаны:
- описательные статистики;
- распределения основных признаков;
- наиболее часто встречающиеся значения;
- корреляции между переменными;
- анализ выбросов;
- визуализация распределений и взаимосвязей.

3. Проверка гипотез:

Были проверены следующие гипотезы:
- Чем выше этаж квартиры, тем выше цена за квадратный метр.
- Чем больше этажей в доме, тем ниже цена за квадратный метр.
- Квартиры с большим количеством комнат имеют более высокую цену за квадратный метр.
- Цена зависит от района расположения квартиры.
- Цена зависит от типа продавца (собственник или агент).

4. Построение модели

Для оценки влияния факторов была построена линейная регрессионная модель.

В модель были включены характеристики квартиры, параметры дома и категориальные признаки, преобразованные в числовой формат. Полученные коэффициенты позволили оценить направление и силу влияния отдельных факторов на стоимость квадратного метра.

**Основные результаты**

В ходе исследования было установлено, что:
- стоимость квадратного метра не увеличивается с ростом этажа квартиры;
- в более высоких домах стоимость квадратного метра в среднем выше;
- количество комнат оказывает статистически значимое влияние на цену, увеличивая ее;
- район является одним из наиболее важных факторов формирования стоимости;
- тип продавца также влияет на цену объекта.

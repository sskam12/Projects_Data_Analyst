# Data Analysis Projects

Этот репозиторий содержит три отдельных проекта, посвященных анализу данных и визуализации.

## 📁 Power BI — Анализ продаж сети магазинов

Проект с интерактивным дашбордом в Power BI. Включает анализ ключевых показателей продаж, динамики выручки, категорий товаров и других бизнес-метрик.

## 📁 Python — Анализ рынка недвижимости (CIAN)

Исследовательский анализ данных объявлений о продаже квартир с платформы CIAN. Рассмотрены характеристики объектов, выявлены закономерности и факторы, влияющие на стоимость недвижимости.

## 📁 Python — Анализ рейтингов фильмов IMDb

Проект посвящен анализу факторов, влияющих на рейтинг фильмов IMDb. Выполнены исследовательский анализ данных, проверка статистических гипотез и построение моделей машинного обучения для прогнозирования категории рейтинга.
